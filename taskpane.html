Task ID Format
Add the Wrike Task ID at the end of your subject line using square brackets:

Meeting notes re: Project Alpha [98765]

The validator accepts any number of digits between [ and ].

If no Task ID is detected, you will see a warning on first Send. Clicking Send a second time will send the email anyway.
